## How to Use These Files

### Quick Start (WordPress.com - Fastest)
1. Read **00-WORDPRESS-SETUP-GUIDE.md** first
2. Create a WordPress.com account (Personal plan recommended)
3. Create 4 new pages in WordPress using the content from files 01-04
4. Set up navigation menu
5. Add contact form using WPForms or Contact Form 7
6. Publish and share

**Time estimate: 2-4 hours**

### Self-Hosted WordPress (More Control)
1. Read **00-WORDPRESS-SETUP-GUIDE.md** for hosting recommendations
2. Set up hosting (Bluehost, SiteGround, etc.) and install WordPress
3. Choose a clean, professional theme
4. Create pages using files 01-04
5. Install form and SEO plugins
6. Publish and launch

**Time estimate: 1-2 days**

## What to Customize

Before publishing, update these placeholders in the content files:

- [ ] **Contact email/phone**: Replace `[contact@powwra.org]` with your actual contact info
- [ ] **Mailing address**: Add your organization's mailing address for donations
- [ ] **Payment method**: Set up PayPal, Venmo, or check payment instructions
- [ ] **Organizational status**: Clarify if you're a 501(c)(3), 501(c)(4), or informal association (affects tax-deductibility of donations)
- [ ] **PDF links**: Upload your existing PDFs and link to them from the Legal Argument and Resources pages
- [ ] **Attorney approval**: Have your legal counsel review the Legal Argument page before publishing (recommended)
